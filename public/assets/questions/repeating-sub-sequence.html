Repeating Sub-Sequence

Given a string A, find length of the longest repeating sub-sequence such that the two subsequence donâ€™t have same string character at same position,

Input Format:

The first and the only argument of input contains a string A.

Output Format:

Number of repeating characters

Constraints:

1 <= |A| <= 100

Examples: 
Input 1:
    A = "abab"

Output 1:
    2

Explanation 1:
    "ab" is repeated.

Input 2:
    A = "abba"

Output 2:
    0

Explanation 2:
    There is no repeating subsequence.

public class Solution {
    public int anytwo(String A) {
        int ans = lrs(A, A);
        return  ans >= 2 ? ans : 0;
    }
    public int lrs(String a, String b) {
        int[][] dp = new int[a.length()+1][b.length()+1];
        for (int i=0; i<=a.length(); i++) {
            for (int j=0; j<=b.length(); j++) {
                if (i == 0 || j == 0)
                    dp[i][j] = 0;
                else if (a.charAt(i-1) == b.charAt(j-1) && i != j)
                    dp[i][j] = dp[i-1][j-1]+1;
                else
                    dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1]);
            }
        }
        return dp[a.length()][b.length()];
    }
}
